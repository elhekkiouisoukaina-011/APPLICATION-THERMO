# PICthermo

Projet réalisé par **ELHEKKIOUI Soukaina**

## Description
PICthermo est une application web développée en Python (Flask) permettant de simplifier les calculs de thermodynamique, notamment :

- Calcul de la pression de bulle (P_bulle)
- Calcul des fractions molaires vapeur (y1, y2)
- Vérification de la somme y1 + y2 = 1
- Affichage détaillé des calculs
- Historique des calculs avec possibilité de suppression

## Technologies utilisées

- Python 3
- Flask
- SQLite (base de données)
- HTML5

## Bibliothèques utilisées

- Flask
- sqlite3 (intégré à Python)
